# Installation


## Package Managers

#### Just Clone 

```bash
git clone https://github.com/Zhengtq/BFQ.git 
```


    
## Version 

*   Minified Version: v1.2.0.1 


## Usage
```markdown
git clone https://github.com/Zhengtq/BFQ.git 
cd BFQ
mkdir build
cmake ..
make -j8
./examples/lession_11
```
